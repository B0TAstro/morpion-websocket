# Fonctionnalités
- Jeu de morpion à 2 joueurs
- Système de salons
- Système d'invitation
- Possibilité de rejouer avec le même adversaire

## Technologies
- [Node.JS]
- [Bootstrap 5]
- [jQuery]
- [Express JS]
- [Socket.io]

### Installation
```sh
npm install
```

#### Exécuter l'application web
```sh
node app.js
```
